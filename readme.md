# Research Data Collector (Behavioral Analysis)

Ce projet est un outil de collecte de données et d'analyse comportementale conçu pour la recherche universitaire. Il permet d'étudier l'interaction entre les systèmes de décision automatisés et les plateformes de jeux stratégiques.

## Objectifs de recherche
- Analyse de la détection comportementale (heuristiques anti-triche).
- Étude de l'impact des délais de réflexion sur la perception humaine.
- Isolation des interfaces via Shadow DOM pour la protection de la vie privée.

## Installation
1. Accédez à `chrome://extensions`.
2. Activez le **Mode développeur**.
3. Cliquez sur **Charger l'extension non empaquetée**.
4. Sélectionnez ce dossier.

*Note : Cet outil est strictement réservé à un usage de recherche éthique. L'utilisation sur des plateformes publiques doit respecter les conditions d'utilisation des sites tiers et les accords éthiques universitaires.*
